# Технические подробности
## Выгрузка фотографий
- Teleram: штатными средствами из Telegram Desktop (три точки -> Export chat history)
- VK: пока не выгружены

 ## Внешний OCR
 - Использована модель Mistral для OCR по API.
 - Часто вылетали ошибки соединения, пришлось завернуть обработку в ретраи

 ## Парсинг ответов
 - Таблицы результатов удалось распарсить полностью
 - Хэдеры с информацией об играх удалось распарсить в большинстве
 - Около 200 ячеек придётся заполнить вручную
 - После проверки результатов выяснилось что качества Mistral OCR не хватает для задачи
 - Принято решение сделать свою классификацию и OCR

## Классификация фотографий
- Для задачи извлечения признаков был применен метод Hu Moments
- Удалось добиться F1-метрики на уровне 0.98
- При использовании стандартного скейлера ничего не поменялось
- Более подронбно в [ноутбуке](research/classification.ipynb)
